</PlaygroundPreview>

### Principles

- **The NavigationLink component is meant to be used inside the header and footer components!** If you're looking for inline links, see the documentation about <InternalLink href="/components/link">HDS link components</InternalLink>.
- Use the NavigationLink component only inside <InternalLink href="/components/header-universal-bar">HeaderUniversalBar</InternalLink>, Navigation and Footer components.
- Because navigation links are designed for site navigation, they should not open in a new tab. However, this may be desired if there is an external link in the HeaderUniversalBar or the Footer.
- External links are not yet supported
- Icons are not yet supported
