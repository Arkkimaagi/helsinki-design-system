## Customisation

### Customisation properties
The component property `setDateClassNames` can be used to set class names for specific dates.

### Customisation example
In this example some days have different background colours to indicate that they have limited timeslots available. When days have colours to indicate something, it's important to include `legend` and `setDateAriaDescribedBy` props to inform the users of their meaning. Use the `setDateAriaDescribedBy` callback function to return corresponding LegendItem's element id for specific dates.  

<Playground scope={{ DateInput, useState }}>


  ```jsx
import { DateInput } from 'hds-react';

() => {
    const dayHasSomeSpace = (day) => day === 12 || day === 15 || day === 24 || day === 25;
    const dayHasLittleSpace = (day) => day === 3 || day === 4 || day === 17 || day === 19;

    const setDateClassNames = (date) => {
        const day = date.getDate();
        if (dayHasSomeSpace(day)) return 'some-space-left';
        if (dayHasLittleSpace(day)) return 'little-space-left';
        return undefined;
    };

    const setDateAriaDescribedBy = (date) => {
        const day = date.getDate();
        if (dayHasSomeSpace(day)) return 'some-space-left';
        if (dayHasLittleSpace(day)) return 'little-space-left';
        return undefined;
    };
    return (
    <>
      <style>
        {`
          .some-space-left {
              background: var(--color-tram-medium-light);
          }

          .little-space-left {
            background: var(--color-summer-medium-light);
          }
        `}
      </style>
      <DateInput
        initialMonth={new Date()}
        onChange={function noRefCheck(){}}
        helperText="Custom styles for days with limited available timeslots in the date picker calendar. Use format D.M.YYYY."
        setDateClassNames={setDateClassNames}
        setDateAriaDescribedBy={setDateAriaDescribedBy}
        legend={[
          { id: 'some-space-left', label: 'Some free timeslots available', color: '#a3e3c2' },
          {
            id: 'little-space-left',
            label: 'Only a few free timeslots available',
            color: '#ffe49c',
          },
        ]}
        style={{ width: 400 }}
      />
    </>
  );
}
```
