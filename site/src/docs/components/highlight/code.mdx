### Properties

Note! You can find the full list of properties in the <Link openInNewTab openInNewTabAriaLabel="Opens in a new tab" href="/storybook/react/?path=?path=/story/components-highlight--default-highlight">React Storybook.</Link>

| Property  | Description                     | Values                        | Default value |
| --------- | ------------------------------- | ----------------------------- | ------------- |
| `text`    | Text for the Highlight/quote    | -                             | -             |
| `size` | Highlight size               | `"s"` `"m"` `"l"`             | `"m"`         | 
| `type`    | Type of the Highlight           | `"highlight"` `"quote"`       | `"highlight"` |
