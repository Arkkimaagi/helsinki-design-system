## Accessibility

### Pay attention to

- Koros are decorative items and should be used in background and hidden from screen readers. See e.g. <InternalLink href="/components/hero">Hero</InternalLink> and <InternalLink href="/components/footer">Footer</InternalLink>.
- **Note that in this documentation the koros have alt texts as they present the component and are therefore important content.**
- Koros are not interactive elements and do not receive focus.
