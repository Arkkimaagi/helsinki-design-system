#### Custom NavigationLink

- The Footer.NavigationLink component has the `as` prop which you can use to use your own navigation link instead of our Footer.NavigationLink, which uses <InternalLink href="/components/">HDS Link</InternalLink> underneath. You can pass it as an element `as={<CustomLink />}` or as a component `as={CustomLink}`.

<Playground>

```jsx
import { Footer, IconFacebook, IconTwitter, IconInstagram, IconYoutube, IconTiktok } from 'hds-react';
import { Link } from 'gatsby';

() => (
  <Footer title="Helsinki Design System">
    <Footer.Navigation>
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
      <Footer.NavigationLink as={Link} label="Item" />
    </Footer.Navigation>
    <Footer.Utilities
      soMeLinks={[
        <Footer.NavigationLink as={Link} title="Helsingin kaupungin Facebook-tili" icon={<IconFacebook />} />,
        <Footer.NavigationLink as={Link} title="Helsingin kaupungin Twitter-tili" icon={<IconTwitter />} />,
        <Footer.NavigationLink as={Link} title="Helsingin kaupungin Instagram-tili" icon={<IconInstagram />} />,
        <Footer.NavigationLink as={Link} title="Helsingin kaupungin Youtube-tili" icon={<IconYoutube />} />,
        <Footer.NavigationLink as={Link} title="Helsingin kaupungin Tiktok-tili" icon={<IconTiktok />} />,
      ]}
    >
      <Footer.NavigationLink as={Link} label="Contact us" />
      <Footer.NavigationLink as={Link} label="Give feedback" />
    </Footer.Utilities>
    <Footer.Base copyrightHolder="Copyright" copyrightText="All rights reserved" backToTopLabel="Back to top">
      <Footer.NavigationLink as={Link} label="Link" />
      <Footer.NavigationLink as={Link} label="Link" />
      <Footer.NavigationLink as={Link} label="Link" />
    </Footer.Base>
  </Footer>
)
```
