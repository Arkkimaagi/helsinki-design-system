# Placeholders

<LargeParagraph>Placeholders can be used to replace missing content or denote areas left intentionally blank.</LargeParagraph>

Examples of possible use cases for placeholders:
- In place of a missing image for example in a search listing blog posts, news items or events.
- As a default image preview for an user uploaded image.

## Usage
Default HDS placeholders include **image placeholders** and **user image placeholders** for different sizes.
