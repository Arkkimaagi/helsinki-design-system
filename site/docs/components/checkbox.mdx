## Disabled

<Playground>
  <Checkbox id="checkbox3" labelText="Label" disabled />
  <Checkbox id="checkbox4" labelText="Label" checked disabled />
</Playground>

##### Core:
```html
<!-- Disabled -->
<div class="hds-checkbox">
  <input type="checkbox" id="checkbox" class="hds-checkbox__input" disabled />
  <label for="checkbox"  class="hds-checkbox__label">Label</label>
</div>

<!-- Selected & disabled -->
<div class="hds-checkbox">
  <input type="checkbox" id="checkbox" class="hds-checkbox__input" checked disabled />
  <label for="checkbox"  class="hds-checkbox__label">Label</label>
</div>
```

##### React:
```tsx
/* Disabled */
<Checkbox id="checkbox" labelText="Label" disabled />

/* Selected & disabled */
<Checkbox id="checkbox" labelText="Label" checked disabled />
```
