# Dropdowns

<LargeParagraph>
  Dropdown offers user a list of options which one or multiple can be selected. Dropsdowns are often used as part of forms and filters.

## Usage

### Default

<Playground>
<Dropdown
  id="example1"
  options={[
    { label: 'Plutonium' },
    { label: 'Americium' },
    { label: 'Copernicium' },
  ]}
  placeholder="Choose one"
  label="Label"
  helper="Assistive text"
/>
</Playground>

