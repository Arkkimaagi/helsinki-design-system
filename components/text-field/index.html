Text Fields

A text field is an input field that the user can interact with and input content and data. This component can be used for long and short form entries. It has a range of options and supports several text formats including numbers. Text fields are normally found within a form but can also be part of a modal or search. Common text input types include: usernames, descriptions, URLs, emails, addresses, and plain text searches.

How to use

  • Use a text input when a single line of text is expected. Use a text area for more than one sentence.
  • Where possible, add programmatic assistance. Detect and pre-fill inputs to reduce errors and save time. Use sentence-case for default values, detected values, and auto-completion text.

Size

Allow the text input to reflect the length of the content you expect the user to enter.
